 # What is this?

 Apply Shadow, Padding, Margin, border, blurness to your pictures.

 # Installation

 `npm i picturedecor --save`

 Then...

```
 import { picturedecor } from 'picturedecor';

 picturedecor ({
     shadow: true;
     padding: true;
     border: true;
 });
```

# Options

picturedecor support 3 options

* shadow, padding, border (default to true)
