 # What is this?

 Apply Shadow, Padding, Margin, border, blurness to your pictures.

 # Installation

 'npm i PictureEffects --save'

 Then...

```
 import { PictureEffects } from 'pictureEffects';

 PictureEffects ({
     shadow: true;
     padding: true;
     border: true;
 });
```

# Options

PictureEffects support 3 options

* shadow, padding, border (default to true)
